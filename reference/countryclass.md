# World Bank Country and Lending Groups

This dataset is produced by the World Bank Group to classify countries
as to their income levels and other groups.

## Usage

``` r
countryclass
```

## Format

\## \`countryclass\` A data frame with 267 rows and 4 columns:

- country_code:

  World Bank country code

- economy:

  Country name

- region:

  World Bank region

- income_group:

  World Bank income classification

## Source

\<https://ddh-openapi.worldbank.org/resources/DR0095333/download/\>
