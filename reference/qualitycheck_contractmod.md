# Quality Checks for Harmonized HRMIS Contract Data

Quality Checks for Harmonized HRMIS Contract Data

## Usage

``` r
qualitycheck_contractmod(contract_tbl, required_vars = NULL)
```

## Arguments

- contract_tbl:

  A harmonized HRMIS contract tibble (e.g. contract_alagoas_tbl)

- required_vars:

  A character vector of required variable names (from dictionary)

## Value

A pointblank agent with a battery of test results
